# Legacy Files

This directory contains code and prompts that are no longer actively used but preserved for reference.

## Code (`legacy/code/`)
- `google_agent.py` - Original Google agent implementation (replaced by inference.py)
- `dummy2.py` - Experimental math tools implementation

## Prompts (`legacy/prompts/`)
- `google_agent.txt` - Original Google agent prompt
- `basic_v0.txt` - Initial basic math solver prompt
- `critic_pattern_v1.txt` - First version of pattern analysis prompt (replaced by v2)
- `updater_v1.txt` - First version of updater prompt (replaced by v2)

These files are kept for historical reference and may be useful for understanding the evolution of the system.
